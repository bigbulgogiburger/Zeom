# 화면기획서 v2 (상세, 웹+앱 통합) — 천지연꽃신당

> 중요 원칙: 화면 품질이 매출/신뢰/재방문을 결정한다.  
> 본 문서는 UX 흐름, 상태, 예외, 정책 문구까지 포함한 실무용 상세 화면기획서다.

---

## 0. UX 핵심 원칙

1. **예약/결제/입장 3클릭 내 완결**
2. **60분 상담 규칙을 모든 화면에서 일관 표기**
3. **불안감 제거 UX**: 결제·환불·노쇼 정책을 화면마다 명확히 노출
4. **신비로운 한국적 톤** + 과한 장식 지양(가독성 우선)
5. **모든 CTA는 상태 기반** (가능/불가 이유를 명확히)

## 2. 화면 상세 정의

## S01. 홈
### 목적
- 사용자가 "지금 무엇을 하면 되는지" 즉시 이해

### 핵심 컴포넌트
- 오늘의 운세 티저 카드
- 추천 상담사 캐러셀
- 오늘 가능한 슬롯 모듈
- 이벤트 배너
- 빠른 CTA: `상담 예약`, `지금 상담 가능`, `캐시 충전`

### 상태
- 로그인 전: 상담 찾기 유도, 회원가입 CTA 강조
- 로그인 후: 개인화(최근 본 상담사/보유 캐시)

### 이벤트
- 추천 상담사 클릭률
- 홈→예약 전환율

---

## S02. 상담사 목록
### 목적
- 비교/탐색 피로 최소화

### 필터
- 카테고리(연애/금전/진로/가족/종합)
- 방식(화상/음성)
- 가능 시간(지금/오늘/이번주)
- 가격(1시간 캐시)
- 평점

### 카드 정보
- 프로필 이미지
- 이름/한줄소개
- 평점/후기수
- 다음 가능 슬롯
- 1시간 필요 캐시

### 예외
- 결과 없음: 필터 초기화 버튼 + 추천 상담사 제안

---

## S03. 상담사 상세
### 목적
- 예약 확정 결정 지원

### 섹션
- 상담사 소개/전문분야
- 후기 리스트
- 예약 슬롯(달력+시간)
- 채널 선택(화상/음성)

### 정책 문구(필수)
- "상담은 1회 60분 단위로 진행됩니다."
- "예약 확정 후 취소 시점에 따라 환불 정책이 적용됩니다."

### CTA
- `선택한 시간으로 예약`

---

## S04. 예약 확인
### 목적
- 예약 실수 방지

### 표시 정보
- 상담사/날짜/시간/채널
- 필요 캐시/보유 캐시
- 부족 캐시 계산값

### 분기
- 캐시 충분: 예약 확정 가능
- 캐시 부족: `캐시 충전 후 예약` CTA

### 유효성
- 슬롯 만료(선점됨) 시 즉시 안내 + 슬롯 재선택

---

## S05. 캐시 충전/결제
### 목적
- 이탈 없는 결제 완료

### 요소
- 상품 카드(60분권, 패키지)
- 결제수단
- 결제 약관 동의
- 결제 진행 버튼

### 결제 처리
- PortOne 호출
- KG이니시스 승인
- 성공 시 지갑 반영 + 예약 화면 복귀

### 실패 UX
- 실패 이유 요약
- 재시도/다른 수단 버튼

---

## S06. 예약/일정
### 목적
- 예약 관리 단일 허브

### 구성
- 탭: 예정 / 완료 / 취소
- 예약 카드: 상담사, 시간, 상태, 채널
- 상담 시작 버튼(시작 10분 전 노출)

### 상태 라벨
- 예약확정 / 진행중 / 완료 / 취소 / 노쇼

### 취소 액션
- 취소 시 환불 예정 캐시/금액 즉시 미리보기

---

## S07. 대기실
### 목적
- 통화 품질 문제 사전 차단

### 요소
- 카메라/마이크 미리보기
- 네트워크 품질 지표
- 입장 카운트다운
- 상담 유의사항

### 불량 상태
- 마이크 권한 없음
- 네트워크 불량
- 상대 미입장

각 상태별 안내문 + 해결 버튼 제공

---

## S08. 상담실 (Sendbird)
### 목적
- 안정적 60분 상담

### UI
- 영상/음성 영역
- 타이머(남은 시간)
- 컨트롤(음소거/카메라/스피커)
- 종료 버튼

### 시간 이벤트
- 10분 남음 알림
- 1분 남음 알림
- 60분 도달 시 자동 종료 처리

### 예외
- 재연결 안내 (끊김 시)
- 재입장 허용 시간 정책 노출

---

## S09. 상담 완료/후기
### 목적
- 경험 회수 + 재구매 유도

### 요소
- 별점/후기 입력
- 재예약 버튼
- 문제 신고 버튼

### 제약
- 후기는 예약 1건당 1회(수정 가능 기간 정책)

---

## S10. 상담 내역
### 목적
- 신뢰성(기록) 제공

### 정보
- 상담일시, 상담사, 소모 캐시, 상태
- 결제/환불 링크
- 영수증 보기

---

## S11. 고객센터
### 목적
- 분쟁/불안 최소화

### 메뉴
- FAQ
- 1:1 문의
- 환불 정책
- 노쇼/지각 정책

### 필수 문구
- 환불 기준 시간표
- 장애 발생 시 보상 기준

---

## S12. 마이페이지
### 목적
- 계정/보안/알림 관리

### 항목
- 프로필
- 알림 설정
- 결제 내역
- 환불 내역
- 약관/동의 이력
- 회원탈퇴

---

## 3. 웹/앱 차이 적용 가이드

### 웹
- 정보량 많은 비교/리스트 최적화
- 달력/필터 고도화

### 앱
- 빠른 액션 중심(예약/입장)
- 하단탭 + 바텀시트 필터
- 푸시 진입(예약 10분 전 → 대기실)

---

## 4. 디자인 디렉션(한국적·신비)

- 메인 컬러: Midnight Navy / Gold 포인트
- 보조 컬러: 자주빛 포인트, 아이보리 배경
- 카드 라운드 + 은은한 글로우
- 아이콘: 연꽃/별자리 모티프 (과장 금지)
- 타이포: 본문 가독성 우선 + 제목만 분위기체 제한 사용
