## 2) 공통 컴포넌트 스펙

## C-01 Top Navigation (Web)
- 높이: 72
- 좌측: 로고
- 중앙: 메뉴(Home, 상담찾기, 예약/일정, 상담내역, 고객센터)
- 우측: 캐시잔액, 프로필 드롭다운

## C-02 Bottom Tab (App)
- 탭 5개: 홈/상담찾기/예약/내역/마이
- 아이콘 + 라벨
- 활성 탭 골드 포인트

## C-03 Counselor Card
- 구성: 이미지(64), 이름, 태그(전문분야), 평점, 다음 가능 슬롯, CTA
- 상태: 기본/호버/선택/비활성

## C-04 Slot Chip
- 크기: 88x36
- 텍스트: `14:00-15:00`
- 상태: OPEN/BOOKED/SELECTED

## C-05 Policy Notice Box
- 아이콘 + 정책 텍스트
- 사용 화면: 예약확인/결제/취소

## C-06 CTA Button
- Primary / Secondary / Ghost
- 상태: default/hover/pressed/disabled/loading

## C-07 Timer Banner
- 상담실 상단 고정
- 텍스트: `남은 시간 00:59:12`
- 10분/1분 임계치 색상 변경

## 5) 플로우 보드(Flow)

## F-01 예약 성공 플로우
홈 → 상담사목록 → 상세 → 예약확인 → 결제(필요 시) → 예약완료 → 예약탭

## F-02 상담 시작 플로우
예약탭 → 대기실 → 상담실 → 종료 → 후기

## F-03 취소/환불 플로우
예약상세 → 취소 → 환불규정 확인 → 취소완료/환불처리 상태
